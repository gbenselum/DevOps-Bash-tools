# Docker Compose Files

This directory contains Docker Compose files for various services.

---

### `circleci.yml`

Docker Compose file for running a local CircleCI runner.

---

### `concourse.yml`

Docker Compose file for running a local Concourse CI server.

---

### `gerrit.yml`

Docker Compose file for running a local Gerrit code review tool.

---

### `gocd.yml`

Docker Compose file for running a local GoCD server.

---

### `jenkins.yml`

Docker Compose file for running a local Jenkins server.

---

### `keycloak.yml`

Docker Compose file for running a local Keycloak identity and access management server.

---

### `octopus-deploy.env`

Environment file for the Octopus Deploy Docker Compose file.

---

### `octopus-deploy.yml`

Docker Compose file for running a local Octopus Deploy server.

---

### `prometheus.yml`

Docker Compose file for running a local Prometheus monitoring server.

---

### `teamcity.yml`

Docker Compose file for running a local TeamCity server.

---

### `wordpress.yml`

Docker Compose file for running a local WordPress instance.
